MES PROJETS

CarDealer – Logiciel de gestion de véhicule

CarDealer est un logiciel de gestion de véhicules conçu pour faciliter la gestion des informations relatives aux véhicules, aux vendeurs et aux clients. Il permet d'ajouter, de modifier, de supprimer et d'afficher les données concernant les véhicules, les vendeurs et les clients. Toutes les informations sont stockées dans une base de données relationnelle. Ce projet a été réalisé en Java, offrant ainsi une solution robuste et efficace pour les concessionnaires automobiles.
Github CarDealer

GSB – Application de gestion de tickets

GSB est une application de gestion de tickets conçue pour gérer les remboursements de frais professionnels. Elle offre plusieurs types de comptes : utilisateur, gestionnaire de remboursements et administrateur. Développée avec Flutter pour l'interface utilisateur, l'application permet aux utilisateurs de soumettre, afficher, modifier et supprimer leurs tickets de frais. Le back-end, réalisé avec Symfony, propose différents modes d’accès : le mode utilisateur permet de gérer les tickets personnels, tandis que le mode gestionnaire est dédié à l'approbation et au remboursement des tickets. Le mode administrateur offre une gestion complète du système, y compris l’ajout et la gestion des comptes utilisateurs. Cette application centralise et simplifie le processus de gestion des frais professionnels, tout en assurant une expérience utilisateur fluide et une administration efficace.
